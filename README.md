# Memo-Server
 
 
## 서론

**계기**
- 친구가 취업을 위해 프론트엔드 공부를 하면서 사용할 백엔드 서버 개발 부탁

**주제**
- 할 일 관리를 위한 Node.JS 및 DB 구현

**설명**
- 간단한 CRUD 처리를 위해 Node.JS 기반 서버 구축
- MySQL과 연동, mysql 모듈이 아닌 sequelize 모듈을 사용
- RESTFUL한 코드 구성을 지향
- (추가 예정)
